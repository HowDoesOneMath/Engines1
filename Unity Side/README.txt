CONTROLS:
Right-click to enter camera move mode, and drag with the mouse to rotate the camera. WASD + QE to move the camera in space while in move mode.

Left click anywhere to set 3D cursor

Left click on object to select it

Press place to place an object at 3D cursor (must have selected object from bottom bar first)

Press delete while an object is selected to delete the selected object

Press Undo to undo, Redo to redo, Save to save, Load to load

Press modify position/rotation to modify a selected object's position/rotation respectively

Left click once to begin moving/rotating the object, left click again to stop

Press the back button to get out of the move/rotate object menu
